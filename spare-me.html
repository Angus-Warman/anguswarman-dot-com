React is a fundamentually unserious piece of engineering

This example is from the excellent guide to Common Beginner Mistakes in React. Take a look at this:

function App() {
    const [x, setX] = React.useState(0);

    function foo() {
        setX(5);
        console.assert(x == 5);
        // Fails
    }
}

If any other language tried to pull something like this, it'd be laughed out of the room. Yes, I know, rendering and so forth. But look at it.
